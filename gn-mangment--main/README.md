# منظومة إدارة التجهيز - إقليم الحرس الوطني بالمتلوي

هذه المنظومة مصممة لإدارة التجهيزات، الاعتمادات المالية، الوضعيات العقارية، الوسائل والصيانة، والمحروقات.

## التشغيل باستخدام Docker

لتشغيل المنظومة بالكامل (قاعدة البيانات، الخلفية، والواجهة الأمامية) باستخدام Docker، اتبع الخطوات التالية:

1. تأكد من تثبيت **Docker** و **Docker Compose**.
2. من المجلد الرئيسي للمشروع، قم بتشغيل الأمر التالي:
   ```bash
   docker compose up --build
   ```
3. ستكون الواجهة الأمامية متاحة على: `http://localhost:5173`
4. ستكون الخلفية (API) متاحة على: `http://localhost:5000`

## بيانات الدخول الافتراضية (بعد تشغيل Seed)

- **المسؤول العام (ADMIN):** admin@example.com / admin123
- **مدير المستودع (MANAGER):** manager@example.com / manager123
- **مستخدم عادي (USER):** user@example.com / user123

## التقنيات المستخدمة

- **الواجهة الأمامية:** React (Vite), Tailwind CSS, Lucide Icons, Recharts.
- **الخلفية:** Node.js (Express), Prisma ORM.
- **قاعدة البيانات:** PostgreSQL.
- **الحاوية:** Docker & Docker Compose.
